# UX Journal Entry I
## Minesweeper
Minesweeper is a puzzle video game where the goal is to click on every tile that is not a mine. In the game flags can be used to indicate where the player thinks a mine is located. 

Minesweeper has historically been made for computers, however there have been numerous attempts of translating the game to a mobile friendly version. One limitation that arises while trying to port the game from computer to mobile is changing how the user can reveal a tile and how to place a flag to indicate a mine. On computer versions, revealing a tile is done with left clicking and placing a flag is done with a right click. This of course can not be replicated on mobile as there is only input which is the user tapping on the screen. 

The solution many mobile versions implemented is having two different modes, one used to place flags and one to reveal tiles.
